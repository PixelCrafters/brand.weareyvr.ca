WeAreYVR

Brand standards guide v1.0

The WeAreYVR brand is designed to represent Vancouver’s startup community without imposing a certain tone of voice or aesthetic upon members of that community. The brand is attentive, authoritative, collegial, purposeful, transparent, and welcoming; it’s honest but aspirational.

Name

  • Always CamelCase the spelling: WeAreYVR, not Weareyvr
  • Do not abbreviate to WAY
  • Always say Y-V-R, not ‘Vancouver’
  • Singular persona: “WeAreYVR is here...” (not “are here”)

Dos and don’ts

Talking points

The logo has two parts: a data-driven changing doodle and an unchanging wordmark. These stand for the ongoing changes that characterize startups and the unbeatable spirit and unity of a community. When used as a design motif, the doodle can be used alone, and its best in groups that way. When used as a logo, the doodle and wordmark must both be present.

Why a generated logo?

The logo’s doodle is adjusted by live data from the WeAreYVR site. It speaks to how data informs what we do, and reflects the changing, active nature of the startup community.

It looks thin/flimsy/weak/minimal

WeAreYVR is here to serve startups, not upstage them. We have an identity, but we step aside to let startup brands shine.

Why does it change?

Startups come in many shapes and sizes, and they change quickly. Our logo changes to show that no matter how different, all the startups here are part of our community.

Aren’t you worried about people mistaking it?

The doodle is mathematically generated, so there’s an order to all the shapes. Most importantly, the WeAreYVR name doesn’t change so we have consistent and dynamic elements in one logo. It’s unique and super-recognizable even as it changes. Over time the change becomes part of what makes it recognizable.

I don’t like it

That’s fair. We can’t please everyone, though we did a comprehensive research and consultation phase and followed a rigorous design process. The important thing is that WeAreYVR is doing something that a lot of people want and need. Being recognized for useful data that helps grow the Vancouver startup community is more important to us.

I like it

That’s great, there’s a pretty good story behind it. [tell any aspect you think is interesting to the person: community consultation, dynamic logo driven by data, etc].

Icons

How to use

For performance reasons, all icons require the base class .wr-icon and an individual icon class. To use, include wr_icons.css and place the following code just about anywhere. Don’t use icon classes and other classes on the same element; instead, apply icon classes to an empty <span>. Be sure to leave a space between the icon and text for proper padding.

<span class="wr-icon wr-icon-startup"></span>

Icon sizes

The icons are designed on a 16×16px grid, so they look best at 16×16px or 32×32px. Icon sizes are automatically adjusted for various where necessary, but you can use the classes below to set the icon size manually, or use the Less mixin .wr-icon-size(@size) within a style declaration for .wr-icon to set a custom size.

Icon Class Size
       .wr-icon-sm 16×16px
       .wr-icon-md 24×24px
       .wr-icon-lg 32×32px
       .wr-icon-xl 48×48px 
.ridiculous-size {
  .wr-icon {
    .wr-icon-size(128px); // 128×128px
  }
}
